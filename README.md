# LeveSix-Android
Allows easier request permissions on Android 6.0 and above

# Install
In project gradle file:
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

In app gradle file:
```
dependencies {
    ...
    compile 'com.github.meeej:LevelSix-Android:v1.0.0'
}
```
